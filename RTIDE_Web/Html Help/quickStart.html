Welcome to RT-IDE. To get started, please refer to the following.

If you require more detail please refer to the General Guide or User Manual

New File

To start making a new HTML page, click "New File...". This will open the main editor.

Open File

To open existing files simply click the "Open File..." option to open a file in the editor.

If you wish to view an existing website in the editor, simply click the "Open URL..." link.

Editing

On the main editor screen, on the right is the code editor box and on the left is the web preview.

Any changes you make will render as HTML on the right automatically.

Changes made in the CSS tab will make changes when you click the "Update" Button.


If you wish to preview your changes in another browser, this is accessible from the "Tools" menu.


Saving

To save changes, you can save both your HTML and CSS file from "File -> Save file" and clicking the appropriate option.
